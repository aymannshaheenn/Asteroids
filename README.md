# Asteroids Game (C++ / SFML)

## Overview

This is a classic Asteroids-style arcade game implemented in C++ using SFML for graphics rendering and window management. The project demonstrates basic game development concepts such as real-time movement, collision detection, object pooling, and simple physics.

---

## Features

- Player-controlled spaceship with rotation and thrust-based movement  
- Shooting mechanics with projectiles  
- Random asteroid spawning and movement  
- Collision detection (ship, bullets, asteroids)  
- Score tracking system  
- Increasing difficulty over time  
- Smooth 2D rendering using SFML  

---

## Technologies Used

- C++
- SFML (Simple and Fast Multimedia Library)

## Controls

- Left Arrow: Rotate left  
- Right Arrow: Rotate right  
- Up Arrow: Thrust forward  
- Spacebar: Shoot  
- Escape: Exit game
